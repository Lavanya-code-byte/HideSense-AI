# 🌿 HideSense AI – Resume Skill Gap Analyzer

HideSense AI is an intelligent web application that analyzes resumes and identifies missing skills based on industry requirements. The system extracts text from uploaded resumes and compares it with a predefined skill dataset to generate a skill gap analysis and downloadable report.

This project helps students and job seekers improve their resumes by highlighting the skills required to become job-ready.

---

## 🚀 Features

* Upload resumes in **PDF or DOCX format**
* Automatic **resume text extraction**
* **Skill matching** with predefined dataset
* Identification of **missing skills**
* **Skill gap analysis visualization**
* Automatic **PDF report generation**
* Simple and clean **web interface**

---

## 🧠 How It Works

1. User uploads a resume.
2. The system extracts text from the document.
3. Extracted content is compared with the skills dataset.
4. Existing and missing skills are identified.
5. A skill comparison chart is generated.
6. A detailed report is provided to the user.
